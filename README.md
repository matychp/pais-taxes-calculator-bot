# PAIS Taxes Calculator Bot

Calculate the total price to import a product in Argentina

## How to use:

1. Start a conversation with the bot.
2. Use the command `/calculate` (use the /help command to know how it works).

## External resources

Current convertion from Argentine Peso to United States Dollars is taken from the [BCRA API](https://estadisticasbcra.com/).
